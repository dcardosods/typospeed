Sobre

A idéia deste jogo é baseada no game Typespeed.

O objetivo é digitar corretamente as palavras em movimento, antes que elas cheguem ao final da tela.

As perdas acontecem, quando um palavra cruza totalmente a tela. Com as palavras em movimento, cada palavra digitada incorretamente conta um erro, e cada palavra palavra digitada corretamente conta um acerto.

Foi desenvolvido e é mantido por Daniel Cardoso, Desenvolvedor Front-end, e aluno do curso de Sistemas para Internet na Universidade Feevale. O código fonte está disponível no Github em github.com/dcardosods/typospeed/. Crontribuições são bem-vindas :)!
