Sobre

A idéia deste jogo é baseada no game Typespeed.

O objetivo é digitar corretamente as palavras em movimento, antes que elas cheguem ao final da tela.

As perdas acontecem, quando uma palavra cruza totalmente a tela. Com as palavras em movimento, cada palavra digitada incorretamente conta um erro, e cada palavra digitada corretamente conta um acerto. O jogo termina com 10 perdas.

As palavras usadas são buscadas no Dicionário Aberto, que são abrangidos pela licença Creative Commons Attribution-Share Alike 2.5 Portugal.

Foi desenvolvido e é mantido por Daniel Cardoso, Desenvolvedor Front-end, e aluno do curso de Sistemas para Internet da Universidade Feevale. O código fonte está disponível no Github em github.com/dcardosods/typospeed/. Crontribuições são bem-vindas :)!
